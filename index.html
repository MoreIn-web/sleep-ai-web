### Overview You want an online, “answer-anything” chatbot you can fully customize in code. Here’s a single-file site you can host on GitHub Pages. It fetches live info from free public sources (no keys needed) and falls back gracefully so it feels broad and useful. It’s built to be extended: add or rearrange “providers” to route any query to the right data source. --- ### What you get - Live facts: Wikipedia and DuckDuckGo Instant Answer. - Weather and time by city: Open‑Meteo Geocoding + Forecast, native timezone formatting. - Currency conversion: ExchangeRate.host. - Crypto prices: CoinGecko. - Tech news: Hacker News API. - Math and unit conversions. - Browser info, URL summarizer (CORS-permitting). - Pluggable “providers” architecture so you can add more sources. --- ### How to deploy 1. Create a public repo (e.g., sleep-ai-online), add this as index.html, commit. 2. Repo Settings → Pages → Deploy from a branch, select main and root. 3. Wait a minute for your github.io URL to go live. --- ### Customize - Add a new provider: copy the provider template in the code, set match() and answer(). - Reorder providers: the first match wins; place your preferred sources higher. - Extend commands and NLP: tweak parse patterns inside providers. - Add datasets: host JSON in the repo and fetch it directly (no keys needed). --- ### Full index.html ```html Sleep AI — Online Chatbot
